【二】反安裝方法
----------------------------------------------------------------
方法 A（推薦，最乾淨）— 直接刪掉本地安裝產物：
    rm -rf /Users/apple/Projects/games/pokemon-mezastar/node_modules
    rm -f  /Users/apple/Projects/games/pokemon-mezastar/package-lock.json
  這會移除上面所有套件，不影響機器其他東西。

方法 B（逐一移除，保留其餘相依）：
    cd /Users/apple/Projects/games/pokemon-mezastar
    npm uninstall react react-dom xstate @xstate/react zustand framer-motion \
                  vite @vitejs/plugin-react typescript \
                  @types/react @types/react-dom vitest

方法 C（連整個專案一起移除）：
    rm -rf /Users/apple/Projects/games/pokemon-mezastar

（可選）清掉 npm 全域快取裡這次下載的 tarball（不影響任何已安裝程式，純省磁碟）：
    npm cache clean --force
